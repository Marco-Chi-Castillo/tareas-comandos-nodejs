## Tareas console App

Esta es una aplicación para crear tareas y poder configurarlas
mediante comandos en nodeJS creados con el paquete
yargs.

Ejecutar este comando para reconstruir los módulos de node y poder ejecutar el código

```
npm install
```

### Puede hacer

- Crear tareas
- Actualizar tareas
- listar tareas
- Elimnar tareas
